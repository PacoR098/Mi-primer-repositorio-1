# Manual de uso del archivo README.md

## ¿Qué es un README.md?
El archivo **README.md** es el documento principal de un repositorio en GitHub.  
Es lo primero que ve cualquier persona que entra al proyecto y sirve para explicar
de forma clara **qué hace el proyecto, cómo funciona y cómo se utiliza**.

Está escrito en **Markdown**, un lenguaje de marcado sencillo que permite dar formato
al texto (títulos, listas, enlaces, código, imágenes) sin necesidad de conocimientos
avanzados.

---

## ¿Cómo editar el README.md en GitHub?

1. Acceder al repositorio en GitHub.
2. Abrir el archivo `README.md`.
3. Pulsar el icono del lápiz (**Edit this file**).
4. Escribir el contenido utilizando sintaxis Markdown.
5. Comprobar el resultado en la pestaña **Preview**.
6. Guardar los cambios pulsando **Commit changes**.
